# Projet 
Déploiement de Tomcat et Jenkins sur serveur Ubuntu par script

Ce projet vise à créer un script capable de déployer Jenkins sur un serveur Ubuntu.

# Pré-requis :
- Un serveur ubuntu fonctionnel en accès par pont
- Archive Tomcat + .war jenkins + Script présent dans le même répertoire
- Une bonne dose de patience
- Une licorne (fourni avec le script)

# Contenu
- Une archive apache-tomcat-7.0.75.tar.gz
- Un fichier jenkins45.war
- Un document DIE - Projet de déploiement.docx
- Un script ScriptJenkins.sh
- Un fichier ReadMe.txt (que vous êtes en train de lire)

# Installation
- Le script doit se situer sur le serveur Ubuntu où le déploiement aura lieu
- Pour l'exécuter, utiliser un super-utilisateur et lancer la commande sudo ./{nom du script}
