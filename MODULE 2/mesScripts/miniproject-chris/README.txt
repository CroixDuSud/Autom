# Déploiement Jenkins45 dans Tomcat sur un serveur ubuntu 

This scripts allows the installation and deployement of Jenkins45 in Tomcat on an ubuntu server

## Getting Started

These instructions will get you a copy of the project up and running on your local machine for development and testing purposes. See deployment for notes on how to deploy the project on a live system.

### Prerequisites

Ubuntu 16.04.4 server

Optional (if you have an internet connection: 
	-Tomcat server application 7.0.75
	-Jenkins 2.32.3

### Installing

Copy 'script.sh' on your ubuntu machine (preferably on a new user environment) and execute.
