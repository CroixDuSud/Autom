-- Installation de Jenkins --

Pour commencer :

En suivant ces instructions, vous vous apprêtez à déployer Jenkins dans tomcat.


Pré-requis :

Serveur Ubuntu fonctionnel. Fichiers sources présents (archive tomcat, jenkins45.war) dans /home/nom_utilisateur

Installation :

1- Lancer la console d'administration du serveur
2- Placez-vous dans le répertoire /home/nom_utilisateur ("cd $HOME")
3- Lancez le script d'installation (./install.sh)
3'- En cas d'absence d'autorisation de lancement tapez "chmod +x install.sh
