-- Installation de Jenkins --

Pour commencer :

En suivant ces instructions, vous vous apprêtez à installer Jenkins sur votre serveur Linux.


Pré-requis :

Serveur Ubuntu fonctionnel.

Installation :

1- Lancer la console d'administration du serveur
2- Placez-vous dans le répertoire /home ("cd /home")
3- Lancez le script d'installation (./install.sh)
3'- En cas d'absence d'autorisation de lancement tapez "chmod +x install.sh
